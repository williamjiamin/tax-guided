## Entry: `us-form-5472-calendar-year-2026`

- Proposed route: `/calendar/us`
- Due date: `2026-04-15`
- Rule type: fixed date for calendar-year filer

### Description

Calendar-year Form 5472 package for a domestic reporting corporation or foreign-owned U.S. disregarded entity using a pro forma Form 1120.

### Notes

- Extension available through Form 7004 if filed by the regular due date.
- Base penalty exposure begins at $25,000 for failure to file.

### Sources

## Entry: `wyoming-annual-report-anniversary-rule`

- Proposed route: `/calendar/us`
- Rule type: recurring anniversary rule

### Description

Wyoming annual report is due on or before the first day of the month of the entity's initial filing date, beginning with the first anniversary year.

### Example

- Initial filing date: February 16, 2025
- First annual report due date: February 1, 2026

### Sources

- [Wyoming "What's Next?" guidance](https://sos.wyo.gov/Forms/WyoBiz/What%27s_Next.pdf)

## Entry: `switzerland-vat-quarterly-60-day-rule`

- Proposed route: `/calendar/ch`
- Rule type: recurring post-period rule

### Description

Swiss VAT returns and payment are generally due within 60 days after the reporting period.

### Example

- If the reporting period ends on March 31, the 60-day due date lands on May 30.

### Sources

- [Paying VAT](https://www.estv.admin.ch/estv/en/home/value-added-tax/paying-vat.html)
